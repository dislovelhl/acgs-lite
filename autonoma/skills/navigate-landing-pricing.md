# Navigate to Pricing Page

## Destination
The ACGS pricing page at `/pricing`

## Steps

1. From any page on the landing site, click the "PRICING" link in the top navigation bar
2. Alternatively, navigate directly to `/pricing`
3. Wait for the page to load
4. Verify the heading "The Engine Is Free Forever" is visible
5. Verify four pricing tier cards are displayed

## What You Should See

Four pricing tier cards arranged in a grid:

### Community Tier
- Price: "Free" with "forever" label
- Description: "Full engine, local audit, MACI separation of powers."
- Features include: Complete governance engine, YAML + code-first rules, MACI separation of powers, Local tamper-evident audit trail, Real-time CLI watch mode, Optional Rust acceleration, Community support (GitHub)
- Button text: "pip install acgs" (links to PyPI)

### Pro Tier (highlighted with blue border and "POPULAR" badge)
- Price: "$299" with "/month" label
- Description: "Compliance reports, cloud audit, 3 regulatory frameworks."
- Features include: Everything in Community, 1M validations/month, Real-time observability stream, EU AI Act Article 12 audit logger, Risk classification engine, Compliance gap report (PDF/JSON), Cloud audit sync (30-day retention), Email support (48h SLA)
- Button text: "Start free trial"

### Team Tier
- Price: "$999" with "/month" label
- Description: "All 9 frameworks, SSO, multi-constitution management."
- Features include: Everything in Pro, 10M validations/month, Portable telemetry bundles, OpenTelemetry + Prometheus integration, Multi-constitution (dev/staging/prod), Constitutional change approval workflow, Cloud audit (1-year retention), SSO (SAML/OIDC), Priority support (4h SLA)
- Button text: "Start free trial"

### Enterprise Tier
- Price: "Custom"
- Description: "On-prem, unlimited, dedicated compliance engineer."
- Features include: Everything in Team, Unlimited validations, 1.1M RPS throughput SLA, 3.9us P99 latency guarantee, Custom regulatory frameworks, On-premise/VPC deployment, Dedicated compliance engineer, Quarterly constitutional review, Audit integration (Splunk, Datadog, ELK), Dedicated support (1h SLA)
- Button text: "Contact sales" (mailto link)

Below the cards: "All prices in USD. Annual billing: save 15%."
