# Navigate to Demo Page

## Destination
The ACGS demo page at `/demo`

## Steps

1. Navigate directly to `/demo` (the demo page is not linked from the main navigation bar)
2. Wait for the page to load
3. Verify the heading "Interactive Demo" is visible
4. Verify the "LAUNCH PLAYWRIGHT DEMO" button is visible

## What You Should See

- A full-screen section with a 3D crystal animation in the background
- The heading "Interactive Demo" in large text
- A "LAUNCH PLAYWRIGHT DEMO" button styled as an outlined pill-shaped link
- Clicking the button navigates to `/demo/playwright`

### Playwright Demo Page

The `/demo/playwright` page shows:
- A heading "Playwright E2E" in large text
- A subtitle "Automated Verification Suite"
- The 3D crystal background animation
- This is a placeholder page for the E2E test demonstration suite
