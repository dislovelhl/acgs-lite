## Principles

### P-001: API-First

**Rationale:** All capabilities should be exposed through documented APIs before user-interface-only delivery.

### P-002: Security by Design

**Rationale:** Security controls, authentication, encryption, and threat modelling must be embedded from project inception.

### P-003: Data Minimisation

**Rationale:** Teams must collect only the personal data needed for a stated purpose and delete it when no longer required.
